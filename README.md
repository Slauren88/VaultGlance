# VaultGlance

VaultGlance is a World of Warcraft addon for the Great Vault.

It adds:

- a vault overlay for Dungeons, Raids, and World/Delves
- a minimap button with a summary tooltip
- local tracking for dungeon and delve names
- chat messages for completions and vault upgrades

## AI note

This addon was coded with AI help.
The idea, testing, and changes were done by me.

## Commands

- `/vg` - open or close the Great Vault
- `/vg refresh` - refresh the addon display
- `/vg list` - print a vault summary in chat
- `/vg debug` - print debug information in chat
- `/vg scan` - read dungeon names from the current Great Vault tooltip
- `/vg settings` - open addon settings

## Install

Put the `VaultGlance` folder inside:

`World of Warcraft\_retail_\Interface\AddOns`

## Notes

- Dungeon and delve names are tracked locally.
- If a dungeon name is missing, hovering the Great Vault dungeon tooltip can let the addon import it.
- Delve names do not transfer between computers.
